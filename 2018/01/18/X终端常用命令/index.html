MAC OS X终端常用命令

发表于 2018-01-18
字数统计: 
pwd　　　　　　当前工作目录

cd（不加参数）　　进root

cd（folder）　　进入文件夹

cd ..　　　　　　上级目录

cd ~　　　　　　返回root

cd -　　　　　　返回上一个访问的目录

rm 文件名 　　　　删除

cat 文件名(|less)　　在终端下查看文件

ls　　　　　　　　列出目录下所有文件

cp 文件名 目标目录　　将文件拷贝到目标目录下

~代表root　　如：~/Document/CPP2/

mkdiv　　　　　　新建文件夹

tree -L 1            列出文件目录树形结构(一级目录)（需要brew install tree）
